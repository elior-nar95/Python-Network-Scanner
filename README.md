# Advanced Python Network Scanner

## Overview
A high-performance port scanner built with Python, featuring multithreading for rapid network discovery. This tool was developed as part of my cybersecurity portfolio to demonstrate networking knowledge and automation skills.

## Features
- **Multithreading:** Scans multiple ports simultaneously.
- **Service Identification:** Detects common services (SSH, HTTP, RDP, etc.).
- **Auto-Logging:** Automatically generates a text report of the scan results.
- **Error Handling:** Robust protection against timeouts and connection errors.

## Technical Skills Demonstrated
- Network Protocols (TCP/IP)
- Python Programming
- Linux/WSL Environment
- Documentation & Reporting (QA standard)

## How to use
1. Run `python3 super_scanner.py`
2. Enter the target IP or Hostname.
3. Check the generated `.txt` report for results.
